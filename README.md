# langchain
langchain challenge

Chatbot to simulate AWS queries via moto. Change the input in the messages variable to change response.

#Environment
You will need an .env file with the following

GOOGLE_API_KEY=''
AWS_ACCESS_KEY_ID=''
AWS_SECRET_ACCESS_KEY=''
AWS_SECURITY_TOKEN=''
AWS_SESSION_TOKEN=''
AWS_DEFAULT_REGION=''

#Execute

python3.10 app.py
